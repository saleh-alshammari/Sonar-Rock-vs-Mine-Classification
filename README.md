# Sonar-Rock-vs-Mine-Classification
Machine learning-based underwater sonar classification system for distinguishing rocks and mines using PCA-driven nonlinear classifiers.

## Overview
The system analyzes sonar signal data to classify objects into:
- Rock
- Mine

## Techniques Used
- PCA (Dimensionality Reduction)
- SVM (Best performing model - 97% accuracy)
- StandardScaler
- Multiple ML models comparison

##  Dataset
- UCI Sonar Dataset
- 208 samples
- 60 features

## Results
- Best Model: SVM
- Accuracy: 97%
